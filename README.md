# JavaBoil

Are you tired of writing the same boilerplate code over and over again? Well, worry no more! With JavaBoil, you can generate boilerplate code for Java with just one command: `!jboil`.

## Getting Started

### Prerequisites

- Visual Studio Code
- Java Development Kit (JDK)

### Installation

1. Open Visual Studio Code.
2. Press `Ctrl+Shift+X` (Windows) or `Cmd+Shift+X` (Mac) to open the Extensions tab.
3. Search for "JavaBoil".
4. Click the "Install" button.

### Usage

1. Open a Java file in Visual Studio Code.
2. Place the cursor where you want to generate the boilerplate code.
3. Type `!jboil` and press `Enter`.
4. JavaBoil will generate the boilerplate code for you.

## Features

- Generates boilerplate code for classes, methods, constructors, and more.
- Supports Java 8 and Java 11.
- Customizable settings.
